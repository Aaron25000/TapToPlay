## 1. Getting Started
### 1.1 Requirements
* Latest version of Node.js.
* Python3 installed.

### 1.2 Cloning the Repo
Clone the repo using `git clone https://github.com/Aaron25000/TapToPlay.git`

### 1.3 Frontend Setup
1. Move to the **frontend** directory by using `cd frontend` on Mac/Linux.
2. Install packages using `npm install`.
3. Run the app using `npm start`.
This should open up a web browser page, if not you can visit http://localhost:3000.

### 1.4. Backend Setup
1. Move to the **backend** directory.
2. Install dependacies: `pip install -r requirements.txt`.
3. Fianlly run the server: `python3 app.py`.


## 2. Contributing

### 2.1 Creating a branch
1. Create a new branch with you name, example: **bob-the-builder**.  The command to do this: `git branch bob-the-builder`.
2. Move to your branch: `git checkout bob-the-builder`.
3. You can verify what brach you are on anytime by: `git branch`.

### 2.2 Committing
1. Change to the **TapToPlay** directory.
2. Run: `git add .`.
3. Make a commit: `git commit -m "You message goes here"`.
4. Push code to **your** branch, example: **bob-the-builder**: `git push origin bob-the-builder`

### 2.3 Merging Code
1. After pushing your code to your branch, go to: https://github.com/Aaron25000/TapToPlay.
2. You should see **"Compare & Pull Request"** button, click it!
3. Then click **"Create Pull Request"**, don't worry about writting a description.
4. The Team Coordinator will review your code for merge errors.
