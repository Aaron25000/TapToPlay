# TapToPlay
Team Billy Joel
## Requirements
Latest version of Node.js

## Getting Started
1. Clone the repo using git clone <repo-name>
2. Install packages using npm install
3. Run the app using npm start
This should open up a web browser page, if not you can visit http://localhost:3000.

## Contributing
. Make a clone of the main branch.
. Only push code to your branch that you can create with the command: git branch <your-name>.
. Submit a pull request only when you are happy your code works and the Team Coordinator will review it.
